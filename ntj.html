Servidor Web
Um servidor web é um computador que armazena arquivos que compõem os sites.
Servidor DNS
O servidor DNS são responsáveis pela tradução desses nomes de domínio em endereços de IP númericos, levando-os aos sites corretos.
Domínio
O domínio é o endereço do seu site na internet.
Hospedagem
Hospedagem de sites é um serviço online que permite que o conteúdo do seu site fique acessível a internet.
